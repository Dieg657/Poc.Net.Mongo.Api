# ASP.NET Core Web API with MongoDB Integration

### Requirements

* Docker
* Docker Compose

### Purpose

This project shows how you can do a WebAPI using ASP.NET Core (.NET 9) and configure an objective Healthcheck to measure if Mongo is healthy/unhealthy!

### How to run?

This project contains a docker compose file (_docker-compose.yaml_) with all required infrastructure included Mongo and all env vars previously setted.

Enjoy yourself to do your own modifications on the file! :D

To run this project you only need execute these commands below:
1. > docker compose up
